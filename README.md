# LunaOrbit website

## Setup

Install dependencies:

```bash
npm i
```

## Dev Server

To run the dev server :

```bash
npm run webpack:start
```

You can view the website at http://localhost:3000/.

## Deploy

```bash
npm run webpack:build
```
